# ISCarRental

Информационная система аренды автомобилей на C# WinForms + MS Access.

## Функции
- регистрация и авторизация
- роли пользователей
- просмотр автомобилей
- оформление аренды
- учет оплат
- возврат и отмена аренды

## Стек
- C#
- Windows Forms
- ADO.NET (OleDb)
- Microsoft Access

## Структура БД
Кратко опиши таблицы:
Users, Accounts, Cars, Rentals, Payments, PaymentMethods, RentalsStatus

## Как запустить
1. Открыть `ISCarRental.sln`
2. Настроить путь к базе данных
3. Проверить наличие `Microsoft Access Database Engine`
4. Запустить проект

## Скриншоты
Добавь окна:
- логин
- регистрация
- каталог машин
- мои аренды
- оплата
